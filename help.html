Setting the timer

After you execute the timer.exe file, a console will pop up. You can do "help" to see list of avaiable commands. To set the timer you need to set hours, minutes and seconds first. Do it with "set" command. After entering the command you will need to set hours, minutes and seconds of the program. Do not type any letters or the program will crash.
